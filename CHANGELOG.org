#+title: Changelog

** 0.1.4

- Org capture from the web UI
  - Reads =org-capture-templates= from Emacs and presents them in a modal (=Cmd+N= or sidebar =+= button)
  - Supports =entry= type templates with static file/headline targets
  - Templates with function targets or bodies shown as "Emacs only"
  - Runs org-capture non-interactively with =:immediate-finish=; all =%u=, =%t=, =%<...>= expansions handled by org
  - "+" button in sidebar header for quick access

** 0.1.3

- Secondary sort by scheduled/deadline time within primary sort groups
  - When sorting by priority, tasks with the same priority are sub-sorted by their scheduled time
  - Works across all sort modes (priority, state, category, deadline)

** 0.1.2

- Mobile-friendly responsive layout
  - Sidebar becomes a slide-over drawer with backdrop on small screens
  - Header controls (sort, group) wrap to a second row on narrow viewports
  - Reduced padding and expanded-panel indent on mobile
  - Metadata rows and date pickers wrap gracefully
- Production server mode: Express serves built frontend (=npm start=)
  - Single-process deployment on port 3001, no separate Vite dev server needed
- launchd service plist for running as a background macOS service
- Safe-area / notch support for iOS devices (=viewport-fit=cover=, =env(safe-area-inset-*)=)
- Use =100dvh= for proper mobile browser viewport height
- Touch-action: manipulation on buttons to eliminate tap delay
- Fix Express 5 catch-all route syntax (=*= -> =/{*path}=) for production static serving

** 0.1.1

- Fix checklist rendering: support all three org checkbox states (=[ ]=, =[-]=, =[X]=)
- In-progress =[-]= items render as orange checkboxes with a dash, italic text
- Progress bar shows orange segment for in-progress items alongside green for checked
- Clicking a checkbox cycles: unchecked -> in-progress -> checked -> unchecked

** 0.1

Initial feature-complete release.

- Today/Upcoming/All Tasks views using org-agenda's own machinery
- Clickable TODO states, priorities, scheduled/deadline dates with calendar+time picker
- Inline notes viewing/editing with interactive org checklists
- Org clocking (clock-in, clock-out, live timer banner)
- Multi-level grouping (agenda type, priority, category, state) with collapsible headers
- Calendar event banners, org markup rendering (bold, italic, code, links)
- Light/dark/auto theme switching with proper CSS variable system
- Collapsible sidebar with keyboard shortcut (=Cmd+\=)
